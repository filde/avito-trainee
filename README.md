## Тестовое задание на стажировку авито

### Запуск
1. Собрать образ docker:
```shell
docker build -t avito-trainee:latest .
```
2. Запуск контейнера:
```shell
docker-compose up
```
  - Если используется postgres, поднятый на компьютере, то необходимо в соответствующие переменные в docker-compose.yml 
указать необходимые значения и закомментировать запуск postgres в docker-compose
### Примечания
1. Если при создании команды один из пользователей уже состоит в другой команде, то выдаётся ошибка и команда не создаётся.
Так было сделано, потому что иначе не понятно, как назначать пользователей на новый pr (из какой из команд?)
2. При создании PR нет проверки на существование команды, т.к. есть проверка на существование пользователя. Если он существует,
то его команда тоже.
3. Ошибка об отсутствии кандидатов на ревью выдаётся только при попытке смены пользователя. При создании PR этой ошибки нет,
т.к. в условиях указано, что при доступном количестве кандидатов меньше 2, указывается 1 или 0 участник группы.
4. Добавлены метрики: количество запросов и количество PR`ов. (+ стандартные метрики системы, такие как температура процессора
и др.)
